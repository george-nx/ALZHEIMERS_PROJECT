
# Dataset

The dataset used in this project is the **Alzheimer MRI 4 Classes Dataset**.

## Classes
- MildDemented
- ModerateDemented
- NonDemented
- VeryMildDemented

## Download Dataset

Download the dataset from the link below:

[https://drive.google.com/your_dataset_link](https://drive.google.com/drive/folders/150VFR0kLVaYQ0AiexS7_Xp582T3iO7Dy?usp=drive_link)


## Instructions

1. Download the dataset.
2. Extract the zip file.
3. Place the folder inside the `dataset` directory.

Example structure:

dataset
│
└── Alzheimer_MRI_4_classes_dataset
     ├── MildDemented
     ├── ModerateDemented
     ├── NonDemented
     └── VeryMildDemented
